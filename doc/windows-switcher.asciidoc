== Introduction

This document describes architectural decisions and technologies for implementing a carousel-style window switcher for X11 environments.

.Core Requirements
* X11 support
* Works with any window manager (WM) or without one
* Window thumbnail previews
* Keyboard shortcuts support: Alt+Tab, Alt+Win+Tab, etc.

== Key Technical Decisions

Before starting development, the following architectural decisions must be made:

=== 1. Programming Language and Core X11 Library

[cols="1,2,2"]
|===
| Option | Advantages | Disadvantages

| **Go + `github.com/jezek/xgb`** ⭐
| Pure Go, no CGo, modern API, good documentation
| Requires understanding of X11 protocol

| Go + `github.com/BurntSushi/xgbutil`
| Simplifies working with keysym, EWMH, ICCCM
| Less active development

| Go + CGo + Xlib
| Maximum compatibility, all examples from C
| CGo overhead, complex build
|===

**Recommendation:** `github.com/jezek/xgb` — active development, good extension support.

=== 2. Application Architecture

[cols="1,2,2"]
|===
| Option | When to Use | How It Works

| **Monolithic Application** ⭐
| Most cases (like alttab)
| • Starts once at system startup
• Grabs Alt+Tab globally
• Shows window on keypress
• Terminates on Ctrl+C or logout

| Daemon + GUI Client
| Only if thumbnail caching between calls or different GUIs needed
| • Daemon: continuous monitoring, MRU, cache
• GUI: launched on daemon request

| i3-specific Solution
| Only for i3wm, simplified version
| • i3 IPC instead of X11 events
• Can work without MRU (i3 manages order itself)
|===

**Recommendation:** Monolithic application — simpler, less code, works like alttab.

.Why Monolithic?
* **Simpler:** single executable file, no IPC
* **Faster:** no GUI process startup overhead
* **More Reliable:** no synchronization issues between processes
* **Like alttab:** proven architecture

.When Is Daemon Needed?
* Persistent thumbnail caching (consumes significant memory)
* Separate GUIs in different languages (GTK, Qt, web)
* Complex logic requiring isolation

| No requirements

| Shared memory

|===

**Recommendation:** XComposite as primary method, XGetImage as fallback.

=== 5. Focus Tracking (MRU Order)

[cols="1,2,1"]
|===
| Method | When to Use | Latency

| **PropertyNotify on `_NET_ACTIVE_WINDOW`** ⭐
| EWMH-compatible WM
| 0 ms

| FocusIn events
| Without EWMH or without WM
| 0 ms

| i3 IPC `window::focus`
| Only i3wm
| 0 ms

| Polling `XGetInputFocus`
| Fallback
| 50-100 ms
|===

**Recommendation:** PropertyNotify + FocusIn as fallback, i3 IPC for i3 version.

=== Implementation Technical Details

.Alt+Tab Grabbing
For correct operation on all systems, we grab the combination accounting for NumLock and CapsLock:

[source,go]
----
ignoreMods := []uint16{
    0,                      // without additional modifiers
    uint16(Mod2),           // NumLock
    uint16(ModLock),        // CapsLock
    uint16(Mod2 | ModLock), // NumLock + CapsLock
}
----

.Getting Window List
Uses EWMH `_NET_CLIENT_LIST` instead of `_NET_CLIENT_LIST_STACKING`, as MRU order is not yet implemented at this stage.

.Window Activation
Sends a ClientMessage event with `_NET_ACTIVE_WINDOW` to root window:

[source,go]
----
event := xproto.ClientMessageEvent{
    Format: 32,
    Window: window,
    Type:   netActiveWindow,
    Data:   []uint32{2, 0, 0, 0, 0}, // source=pager
}
----

.Text UI
Uses terminal raw mode for arrow key handling without waiting for Enter. ESC sequences are recognized by pattern `27 91 <code>`.

==== Focus Tracking (pkg/focus/watcher.go)

Tracks active window changes via two methods:

**Primary Method:** PropertyNotify on `_NET_ACTIVE_WINDOW`::
* Subscribes to PropertyChange events on root window
* Zero latency detection of focus changes
* Works with EWMH-compatible WMs

**Fallback Method:** FocusIn events::
* Direct window focus events
* Used when EWMH not available
* Requires per-window event subscription

[source,go]
----
ctx := context.Background()
watcher := focus.NewWatcher(ctx, conn, root, mruList, capturer)

// In event loop:
case xproto.PropertyNotifyEvent:
    watcher.HandlePropertyNotify(e)
case xproto.FocusInEvent:
    watcher.HandleFocusIn(e)  // Fallback
----

=== Technical Challenges & Solutions

==== Challenge 1: XComposite API Usage

**Problem:** `composite.NameWindowPixmap` requires pre-allocated pixmap ID

**Solution:**
[source,go]
----
pixmapID, _ := xproto.NewPixmapId(conn)
composite.NameWindowPixmapChecked(conn, window, pixmapID)
defer xproto.FreePixmap(conn, pixmapID)
----

==== Challenge 2: Icon Pixel Format

**Problem:** `_NET_WM_ICON` uses ARGB, Go image uses RGBA

**Solution:** Manual pixel conversion during parsing:
[source,go]
----
pixel := data[2+y*width+x]
r := uint8((pixel >> 16) & 0xFF)
g := uint8((pixel >> 8) & 0xFF)
b := uint8(pixel & 0xFF)
a := uint8((pixel >> 24) & 0xFF)
----

==== Challenge 3: PropertyNotify Noise

**Problem:** Root window receives many PropertyNotify events

**Solution:** Filter by atom and window:
[source,go]
----
if e.Window != fw.root || e.Atom != fw.netActiveWindow {
    return  // Ignore irrelevant events
}
----

==== Carousel Renderer (pkg/carousel/renderer.go)

2.5D rendering with perspective effect:

[source,go]
----
func Draw3DCarousel(thumbnails []image.Image, selected int,
                     animOffset float64, cfg Config) *image.RGBA {
    dc := gg.NewContext(cfg.Width, cfg.Height)

    // Semi-transparent background
    dc.SetRGBA(0, 0, 0, 0.85)
    dc.Clear()

    for i := range thumbnails {
        offset := float64(i - selected) - animOffset

        // Calculate scale based on distance from center
        scale := cfg.PerspectiveFactor +
                 (1.0-cfg.PerspectiveFactor)/(1.0+math.Abs(offset)*0.5)

        // Position with spacing
        x := centerX + offset*cfg.Spacing*scale

        // Slight arc effect for depth
        y := centerY + math.Abs(offset)*10

        // Rotation for perspective
        rotation := offset * 0.3

        // Draw thumbnail with transformations
        drawThumbnail(dc, thumbnails[i], x, y, scale, rotation)
    }

    return dc.Image().(*image.RGBA)
}
----

**Key Features:**
* Perspective scaling — windows farther from center appear smaller
* Rotation effect — side windows slightly rotated (pseudo-3D)
* Arc positioning — subtle vertical offset for depth perception
* Alpha blending — distant windows more transparent
* Shadow rendering — depth cues for selected window
* Selection indicator — blue glow around active window

func NewWindow(conn *xgb.Conn, root xproto.Window,
               width, height int) (*Window, error) {
    // Create override-redirect window (no WM decorations)
    mask := xproto.CwBackPixel | xproto.CwOverrideRedirect |
            xproto.CwEventMask

    // Center on screen
    x := (screen.WidthInPixels - width) / 2
    y := (screen.HeightInPixels - height) / 2

    // Set window type to dialog
    setWindowProperty("_NET_WM_WINDOW_TYPE",
                      "_NET_WM_WINDOW_TYPE_DIALOG")

    // Always on top
    setWindowProperty("_NET_WM_STATE",
                      "_NET_WM_STATE_ABOVE")

    return window, nil

type Selector struct {
    conn          *xgb.Conn
    windows       []x11.WindowInfo
    selectedIndex int
    window        *carousel.Window
    config        carousel.Config
    animOffset    float64
    resultChan    chan *x11.WindowInfo

func (s *Selector) Show() (*x11.WindowInfo, error) {
    // Prepare thumbnails with placeholders
    thumbnails := s.prepareThumbnails()

    // Show window and render
    s.window.Show()
    s.render(thumbnails)

    // Handle events in goroutine
    go s.handleEvents()

    // Wait for selection or timeout
    select {
    case result := <-s.resultChan:
        s.window.Hide()
        return result, nil
    case <-time.After(30 * time.Second):
        return nil, fmt.Errorf("timeout")
    }
}
----

==== Animation System

Frame-based smooth transitions:

[source,go]
----
func (s *Selector) animateTransition(targetIndex int,
                                     thumbnails []image.Image) {
    duration := 150 * time.Millisecond
    frames := 15
    frameTime := duration / time.Duration(frames)

    direction := 1.0
    if targetIndex < s.selectedIndex {
        direction = -1.0
    }

    // Animate each frame
    for frame := 0; frame <= frames; frame++ {
        progress := float64(frame) / float64(frames)
        s.animOffset = direction * (1.0 - progress)
        s.render(thumbnails)
        time.Sleep(frameTime)
    }

    s.selectedIndex = targetIndex
    s.animOffset = 0.0
}
----

**Animation Parameters:**
* Duration: 150ms (fast but smooth)
* Frame count: 15 frames
* Frame time: ~10ms per frame
* Easing: Linear (simple and predictable)

== X11 Window Switcher Architecture

=== Getting Window List

In X11, there are two main ways to get the window list:

==== EWMH (Extended Window Manager Hints)

Modern WMs support the EWMH specification, which provides:

* `_NET_CLIENT_LIST` — list of all client windows
* `_NET_CLIENT_LIST_STACKING` — list of windows in stacking order (z-order)
* `_NET_ACTIVE_WINDOW` — current active window
* `_NET_WM_NAME` — window name in UTF-8
* `_NET_WM_ICON` — window icon

[source,go]
----
package x11

import (
    "github.com/jezek/xgb"
    "github.com/jezek/xgb/xproto"
)

// GetClientList gets window list via EWMH _NET_CLIENT_LIST_STACKING
func GetClientList(conn *xgb.Conn, root xproto.Window) ([]xproto.Window, error) {
    // Get _NET_CLIENT_LIST_STACKING atom
    atomReply, err := xproto.InternAtom(conn, false,
        uint16(len("_NET_CLIENT_LIST_STACKING")),
        "_NET_CLIENT_LIST_STACKING").Reply()
    if err != nil {
        return nil, err
    }

    // Read property
    prop, err := xproto.GetProperty(conn, false, root,
        atomReply.Atom,        // property
        xproto.AtomWindow,     // type
        0,                     // offset
        ^uint32(0),            // length (maximum)
    ).Reply()
    if err != nil {
        return nil, err
    }

    // Convert bytes to []Window
    windows := make([]xproto.Window, prop.ValueLen)
    for i := uint32(0); i < prop.ValueLen; i++ {
        windows[i] = xproto.Window(
            uint32(prop.Value[i*4]) |
            uint32(prop.Value[i*4+1])<<8 |
            uint32(prop.Value[i*4+2])<<16 |
            uint32(prop.Value[i*4+3])<<24,
        )
    }
    return windows, nil
}

=== Building Window List with MRU (Focus Order)

For window switcher, "Most Recently Used First" (MRU) order is critically important.
`_NET_CLIENT_LIST_STACKING` only provides z-order (stacking order), which doesn't always correspond to usage order.

==== The Problem

[cols="1,1"]
|===
| Stacking Order (z-order) | Focus Order (MRU)

| Which window is on top of which
| Which window was active last

| `_NET_CLIENT_LIST_STACKING`
| No standard atom in EWMH!
|===

WM don't provide a ready-made MRU list — it must be maintained independently.

==== Solution: Sortlist (like in alttab)

The `alttab` project approach: maintain `sortlist` — a doubly-linked list of windows in MRU order — in parallel with `winlist` (display list).

.MRU List Architecture

.Key Operations with MRU List
[source,go]

package mru

import (
    "sync"

    "github.com/jezek/xgb/xproto"
)

// MRUList — thread-safe list of windows in MRU order
type MRUList struct {
    mu    sync.RWMutex
    items []xproto.Window // [0] = most recent
}

// NewMRUList creates new MRU list
func NewMRUList() *MRUList {
    return &MRUList{
        items: make([]xproto.Window, 0),
    }
}

// Touch — move window to front (called on focus)
func (m *MRUList) Touch(windowID xproto.Window) {
    m.mu.Lock()
    defer m.mu.Unlock()

    // Find and remove if already exists
    for i, id := range m.items {
        if id == windowID {
            // Remove from current position
            m.items = append(m.items[:i], m.items[i+1:]...)
            break
        }
    }

    // Add to front
    m.items = append([]xproto.Window{windowID}, m.items...)
}

// Remove — remove window from list (called on close)
func (m *MRUList) Remove(windowID xproto.Window) {
    m.mu.Lock()
    defer m.mu.Unlock()

    for i, id := range m.items {
        if id == windowID {
            m.items = append(m.items[:i], m.items[i+1:]...)
            return
        }
    }
}

// GetOrder — return copy of list in MRU order
func (m *MRUList) GetOrder() []xproto.Window {
    m.mu.RLock()
    defer m.mu.RUnlock()

    result := make([]xproto.Window, len(m.items))
    copy(result, m.items)
    return result
}

type FocusWatcher struct {

    root              xproto.Window
    netActiveWindow   xproto.Atom
    mru               *MRUList

// NewFocusWatcher creates focus watcher
func NewFocusWatcher(conn *xgb.Conn, root xproto.Window, mru *MRUList) (*FocusWatcher, error) {
    // Get _NET_ACTIVE_WINDOW atom
    reply, err := xproto.InternAtom(conn, false,
        uint16(len("_NET_ACTIVE_WINDOW")),
        "_NET_ACTIVE_WINDOW").Reply()
    if err != nil {
        return nil, err
    }

    // Subscribe to PropertyChange events on root window
    err = xproto.ChangeWindowAttributesChecked(conn, root,
        xproto.CwEventMask,
        []uint32{xproto.EventMaskPropertyChange},
    ).Check()
    if err != nil {
        return nil, err
    }

    return &FocusWatcher{
        conn:            conn,
        root:            root,
        netActiveWindow: reply.Atom,
        mru:             mru,
    }, nil
}

// HandlePropertyNotify handles property change event
func (fw *FocusWatcher) HandlePropertyNotify(e xproto.PropertyNotifyEvent) {
    // Only interested in root window and _NET_ACTIVE_WINDOW atom
    if e.Window != fw.root || e.Atom != fw.netActiveWindow {
        return
    }

    // Get current active window
    active, err := fw.GetActiveWindow()
    if err != nil || active == 0 || active == fw.uiWindow {
        return
    }

    // Move to front of MRU list
    fw.mru.Touch(active)
}

// GetActiveWindow returns current active window
func (fw *FocusWatcher) GetActiveWindow() (xproto.Window, error) {
    prop, err := xproto.GetProperty(fw.conn, false, fw.root,
        fw.netActiveWindow, xproto.AtomWindow, 0, 1).Reply()
    if err != nil || prop.ValueLen == 0 {
        return 0, err
    }
    return xproto.Window(
        uint32(prop.Value[0]) |
        uint32(prop.Value[1])<<8 |
        uint32(prop.Value[2])<<16 |
        uint32(prop.Value[3])<<24,
    ), nil
}

import "time"

// PollFocusChanges polls focus every interval
func (fw *FocusWatcher) PollFocusChanges(ctx context.Context, interval time.Duration) {
    ticker := time.NewTicker(interval)
    defer ticker.Stop()

    var lastFocused xproto.Window

    for {
        select {
        case <-ctx.Done():
            return
        case <-ticker.C:
            current, err := fw.GetInputFocus()
            if err != nil {
                continue
            }
            if current != lastFocused && current != 0 {
                fw.mru.Touch(current)
                lastFocused = current
            }
        }
    }
}

==== Sorting Window List by MRU

[source,go]

import "sort"

// SortWindowsByMRU sorts window list by MRU order
func (m *MRUList) SortWindowsByMRU(windows []xproto.Window) []xproto.Window {
    order := m.GetOrder()

    // Create map for fast position lookup in MRU
    orderMap := make(map[xproto.Window]int, len(order))
    for i, id := range order {
        orderMap[id] = i
    }

    // Copy and sort
    sorted := make([]xproto.Window, len(windows))
    copy(sorted, windows)

    sort.Slice(sorted, func(i, j int) bool {
        posI, okI := orderMap[sorted[i]]
        posJ, okJ := orderMap[sorted[j]]

        // Windows not in MRU list go to the end
        if !okI {
            posI = len(order) + 1
        }
        if !okJ {
            posJ = len(order) + 1
        }

        return posI < posJ
    })

    return sorted
}

// PrepareWindowList prepares window list for display
func (wm *WindowManager) PrepareWindowList() ([]WindowInfo, error) {
    // Get window list from WM
    windows, err := GetClientList(wm.conn, wm.root)
    if err != nil {
        return nil, err
    }

    // Sort by MRU
    sorted := wm.mru.SortWindowsByMRU(windows)

    // Fill information about each window
    result := make([]WindowInfo, 0, len(sorted))
    for _, wid := range sorted {
        info, err := wm.GetWindowInfo(wid)
        if err != nil {
            continue // skip problematic windows
        }
        result = append(result, info)
    }

    return result, nil
}

        return nil, err

    if err != nil {
        return 0, err

        return 0, err

// GetWindowImage gets window image as image.Image
func (tc *ThumbnailCapture) GetWindowImage(window xproto.Window) (image.Image, error) {
    // Get window geometry
    geom, err := xproto.GetGeometry(tc.conn, xproto.Drawable(window)).Reply()
    if err != nil {
        return nil, err
    }

    // Capture image
    img, err := xproto.GetImage(tc.conn, xproto.ImageFormatZPixmap,
        xproto.Drawable(window),
        0, 0, geom.Width, geom.Height,
        0xFFFFFFFF).Reply()
    if err != nil {
        return nil, err
    }

    // Convert to image.RGBA
    return tc.bytesToImage(img.Data, int(geom.Width), int(geom.Height), int(geom.Depth))
}

// bytesToImage converts X11 data to image.Image
func (tc *ThumbnailCapture) bytesToImage(data []byte, width, height, depth int) (image.Image, error) {
    img := image.NewRGBA(image.Rect(0, 0, width, height))

    // For 24/32-bit depth (BGRA → RGBA)
    for y := 0; y < height; y++ {
        for x := 0; x < width; x++ {
            i := (y*width + x) * 4
            if i+3 < len(data) {
                // X11 uses BGRA
                img.Pix[(y*width+x)*4+0] = data[i+2] // R
                img.Pix[(y*width+x)*4+1] = data[i+1] // G
                img.Pix[(y*width+x)*4+2] = data[i+0] // B
                img.Pix[(y*width+x)*4+3] = data[i+3] // A
            }
        }
    }

    return img, nil
}

    "github.com/jezek/xgb/shm"

// ShmCapture — capture via shared memory
type ShmCapture struct {
    conn   *xgb.Conn
    seg    shm.Seg
    shmid  int
    data   []byte
}

// NewShmCapture creates capturer via shared memory
func NewShmCapture(conn *xgb.Conn, width, height int) (*ShmCapture, error) {
    if err := shm.Init(conn); err != nil {
        return nil, err
    }

    size := width * height * 4

    shmid, err := syscall.Shmget(syscall.IPC_PRIVATE, size, syscall.IPC_CREAT|0777)

    }

    seg, _ := shm.NewSegId(conn)
    shm.Attach(conn, seg, uint32(shmid), false)

    return &ShmCapture{
        conn:  conn,
        seg:   seg,
        shmid: shmid,
        data:  data,
    }, nil

==== XGrabKey — Global Grab

[source,go]

package keygrab

import (
    "github.com/jezek/xgb"
    "github.com/jezek/xgb/xproto"
)

// Key modifiers
const (
    ModAlt   = xproto.ModMask1 // Alt = Mod1
    ModSuper = xproto.ModMask4 // Super/Win = Mod4
    ModShift = xproto.ModMaskShift
)

// KeyGrabber grabs key combinations
type KeyGrabber struct {
    conn    *xgb.Conn
    root    xproto.Window
    tabCode xproto.Keycode
    altCode xproto.Keycode
}

// NewKeyGrabber creates new key grabber
func NewKeyGrabber(conn *xgb.Conn, root xproto.Window) (*KeyGrabber, error) {
    // Get keycode for Tab
    setup := xproto.Setup(conn)
    tabCode := keysymToKeycode(conn, setup, 0xFF09) // XK_Tab
    altCode := keysymToKeycode(conn, setup, 0xFFE9) // XK_Alt_L

    return &KeyGrabber{
        conn:    conn,
        root:    root,
        tabCode: tabCode,
        altCode: altCode,
    }, nil
}

// GrabAltTab grabs Alt+Tab and Alt+Shift+Tab
func (kg *KeyGrabber) GrabAltTab() error {
    // Get masks for ignored modifiers (NumLock, CapsLock, ScrollLock)
    ignoredMasks := []uint16{0, xproto.ModMaskLock, xproto.ModMask2,
                             xproto.ModMaskLock | xproto.ModMask2}

    for _, ignored := range ignoredMasks {
        // Alt+Tab
        err := xproto.GrabKeyChecked(kg.conn, true, kg.root,
            ModAlt|ignored, kg.tabCode,
            xproto.GrabModeAsync, xproto.GrabModeAsync).Check()
        if err != nil {
            return err
        }

        // Alt+Shift+Tab (reverse)
        err = xproto.GrabKeyChecked(kg.conn, true, kg.root,
            ModAlt|ModShift|ignored, kg.tabCode,
            xproto.GrabModeAsync, xproto.GrabModeAsync).Check()
        if err != nil {
            return err
        }

        // Alt+Super+Tab (additional combination)
        err = xproto.GrabKeyChecked(kg.conn, true, kg.root,
            ModAlt|ModSuper|ignored, kg.tabCode,
            xproto.GrabModeAsync, xproto.GrabModeAsync).Check()
        if err != nil {
            return err
        }
    }

    return nil
}

// UngrabAltTab releases grab
func (kg *KeyGrabber) UngrabAltTab() {
    xproto.UngrabKey(kg.conn, kg.tabCode, kg.root, xproto.ModMaskAny)
}

// IsAltPressed checks if Alt is pressed
func (kg *KeyGrabber) IsAltPressed() bool {
    reply, err := xproto.QueryKeymap(kg.conn).Reply()
    if err != nil {
        return false
    }

    // Check bit for altCode
    byteIndex := int(kg.altCode) / 8
    bitMask := byte(1 << (uint(kg.altCode) % 8))
    return reply.Keys[byteIndex]&bitMask != 0
}

// HandleKeyPress handles keypress
func (kg *KeyGrabber) HandleKeyPress(e xproto.KeyPressEvent, switcher *Switcher) {
    if e.Detail != kg.tabCode {
        return
    }

    if e.State&ModAlt == 0 {
        return // Alt not pressed
    }

    if !switcher.IsVisible() {
        // First press — show switcher
        backward := e.State&ModShift != 0
        switcher.Show(backward)
    } else {
        // Repeated press — switch window
        if e.State&ModShift != 0 {
            switcher.Previous()
        } else {
            switcher.Next()
        }
    }
}

// HandleKeyRelease handles key release
func (kg *KeyGrabber) HandleKeyRelease(e xproto.KeyReleaseEvent, switcher *Switcher) {
    // If Alt is released — activate selected window
    if e.Detail == kg.altCode && switcher.IsVisible() {
        switcher.ActivateSelected()
        switcher.Hide()
    }
}

// keysymToKeycode converts keysym to keycode
func keysymToKeycode(conn *xgb.Conn, setup *xproto.SetupInfo, keysym uint32) xproto.Keycode {
    min := setup.MinKeycode
    max := setup.MaxKeycode

    mapping, err := xproto.GetKeyboardMapping(conn, min, byte(max-min+1)).Reply()
    if err != nil {
        return 0
    }

    keysymsPerKeycode := int(mapping.KeysymsPerKeycode)
    for i := 0; i < int(max-min+1); i++ {
        for j := 0; j < keysymsPerKeycode; j++ {
            if mapping.Keysyms[i*keysymsPerKeycode+j] == xproto.Keysym(keysym) {
                return xproto.Keycode(int(min) + i)
            }
        }
    }
    return 0
}

}

// ActivateWindow activates specified window via EWMH
func (wa *WindowActivator) ActivateWindow(window xproto.Window) error {
    // Form ClientMessage for _NET_ACTIVE_WINDOW
    data := xproto.ClientMessageDataUnionData32New([]uint32{
        2,           // source: 2 = pager
        0,           // timestamp (CurrentTime)
        0,           // currently active window
        0, 0,
    })

    event := xproto.ClientMessageEvent{
        Response_type: xproto.ClientMessageEvent{}.Response_type,
        Format:        32,
        Window:        window,
        Type:          wa.netActiveWindow,
        Data:          data,
    }

    // Send event to root window
    return xproto.SendEventChecked(wa.conn, false, wa.root,
        xproto.EventMaskSubstructureNotify|xproto.EventMaskSubstructureRedirect,
        string(event.Bytes())).Check()
}

// ActivateWindowFallback activates window without EWMH (for old WMs)
func (wa *WindowActivator) ActivateWindowFallback(window xproto.Window) error {
    // Raise window to top
    err := xproto.ConfigureWindowChecked(wa.conn, window,
        xproto.ConfigWindowStackMode,
        []uint32{xproto.StackModeAbove}).Check()
    if err != nil {
        return err
    }

    // Set input focus
    return xproto.SetInputFocusChecked(wa.conn, xproto.InputFocusPointerRoot,
        window, xproto.TimeCurrentTime).Check()
}

// SwitchToWindow switches to window with fallback
func (wa *WindowActivator) SwitchToWindow(window xproto.Window) error {
    // First try EWMH
    err := wa.ActivateWindow(window)
    if err != nil {
        // Fallback for old WMs
        return wa.ActivateWindowFallback(window)
    }
    return nil
}

== Recommendations

=== For Minimal Solution (like alttab)

*Language:* C
*Stack:* Xlib + XComposite + Cairo

This is the `alttab` project path. Minimal dependencies, full control, but requires deep understanding of X11.

=== For Rapid Development with Beautiful UI

*Language:* C or Go
*Stack:* GTK3

GTK is well integrated with X11, supports CSS styles, ready-made widgets for working with Pixbuf.

=== For Maximum Flexibility and Animations

*Language:* C++
*Stack:* Qt5/Qt6 + QML

Qt provides powerful animation and rendering system. QML allows declaratively describing carousel.

=== For Go Project (like i3qws)

*Language:* Go
*Stack:* xgb/xgbutil + separate GUI process

Recommended to split into two components:

. *Go Daemon* — tracks windows, handles keys via xgb
. *GUI Process* — lightweight renderer (can even be GTK/Qt), launched on demand

This avoids CGo overhead for continuous monitoring.

.Go + gg (2.5D graphics)
[source,go]

package carousel

import (
    "image"
    "github.com/fogleman/gg"
)

// Draw3DCarousel draws 2.5D carousel with perspective effect
func Draw3DCarousel(thumbnails []image.Image, selected int, width, height int) *image.RGBA {
    dc := gg.NewContext(width, height)

    // Background
    dc.SetRGBA(0, 0, 0, 0.8)
    dc.Clear()

    centerX := float64(width) / 2
    centerY := float64(height) / 2

    for i, thumb := range thumbnails {
        // Position relative to center
        offset := float64(i - selected)

        // Transformation parameters
        var scale, x, alpha float64
        if i == selected {
            // Central window — full size
            scale = 1.0
            x = centerX
            alpha = 1.0
        } else {
            // Side windows — reduced with perspective
            perspective := 0.6
            scale = perspective + (1.0-perspective)/(1.0+abs(offset)*0.5)
            x = centerX + offset*150*scale
            alpha = 0.7
        }

        // Thumbnail size
        thumbW := float64(thumb.Bounds().Dx()) * scale
        thumbH := float64(thumb.Bounds().Dy()) * scale

        // Apply transformations
        dc.Push()
        dc.Translate(x, centerY)
        dc.Scale(scale, scale)
        dc.Translate(-thumbW/2, -thumbH/2)

        // Draw thumbnail with alpha channel
        dc.SetRGBA(1, 1, 1, alpha)
        dc.DrawImage(thumb, 0, 0)

        dc.Pop()
    }

    return dc.Image().(*image.RGBA)
}

func abs(x float64) float64 {
    if x < 0 {
        return -x
    }
    return x
}

== Handling Without Window Manager

If WM is absent:

. `_NET_CLIENT_LIST` will be unavailable — use `XQueryTree`
. Filter windows by `override_redirect`, `map_state`
. Activation via `XSetInputFocus` + `XRaiseWindow`
. No guarantee of stacking order

[source,go]
----
package windowfilter

import (
    "github.com/jezek/xgb"
    "github.com/jezek/xgb/xproto"
)

// FilterNormalWindows filters windows, leaving only regular applications
func FilterNormalWindows(conn *xgb.Conn, windows []xproto.Window) []xproto.Window {
    result := make([]xproto.Window, 0, len(windows))

    for _, win := range windows {
        if IsNormalWindow(conn, win) {
            result = append(result, win)
        }
    }

    return result
}

// IsNormalWindow checks if window is a regular application
func IsNormalWindow(conn *xgb.Conn, win xproto.Window) bool {
    // Get window attributes
    attrs, err := xproto.GetWindowAttributes(conn, win).Reply()
    if err != nil {
        return false
    }

    // Check 1: window must be visible
    if attrs.MapState != xproto.MapStateViewable {
        return false
    }

    // Check 2: not override_redirect (not popup/tooltip/menu)
    if attrs.OverrideRedirect {
        return false
    }

    // Check 3: WM_STATE must be NormalState
    wmState, err := GetWMState(conn, win)
    if err != nil || wmState != WMStateNormal {
        return false
    }

    return true
}

// WM_STATE constants
const (
    WMStateWithdrawn = 0
    WMStateNormal    = 1
    WMStateIconic    = 3
)

// GetWMState gets window's WM_STATE
func GetWMState(conn *xgb.Conn, win xproto.Window) (uint32, error) {
    // Get WM_STATE atom
    atom, err := xproto.InternAtom(conn, false,
        uint16(len("WM_STATE")), "WM_STATE").Reply()
    if err != nil {
        return 0, err
    }

    // Read property
    prop, err := xproto.GetProperty(conn, false, win,
        atom.Atom,     // property
        atom.Atom,     // type
        0,             // offset
        2,             // length (WM_STATE — 2 longs)
    ).Reply()

    if err != nil || prop.ValueLen == 0 {
        return 0, err
    }

    // First long — state
    state := uint32(prop.Value[0]) |
        uint32(prop.Value[1])<<8 |
        uint32(prop.Value[2])<<16 |
        uint32(prop.Value[3])<<24

    return state, nil
}

=== Fallback for Systems Without Compositor

**Detection:** Checks if compositor is running via XComposite extension

**Fallback Strategy:**

. Try XComposite thumbnail capture first
. On failure, use placeholder with window icon
. Load icon from `_NET_WM_ICON` property
. If no icon available, find by WM_CLASS in desktop files
. Ultimate fallback: generic placeholder

**Implementation Status:**

* ✅ Automatic fallback to placeholders
* ✅ Icon loading from `_NET_WM_ICON`
* ✅ Icon search in desktop files by WM_CLASS
* ✅ Icon search in standard directories (`/usr/share/icons`, etc.)
* ✅ Graceful degradation without warnings

**Desktop File Parsing:**

Searches for `.desktop` files in:

* `/usr/share/applications`
* `/usr/local/share/applications`
* `~/.local/share/applications`
* `$XDG_DATA_DIRS/applications`

Matches by:

. `StartupWMClass` field (exact match)
. `Name` field (case-insensitive)
. Desktop filename (fuzzy match)
